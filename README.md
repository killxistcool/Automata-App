## :grey_question: FAQ

- Is Automata free to use?
- Yes, and it’ll most likely stay that way.
- Is Automata malware?
- No, it doesn’t contain any malware.
- What does Automata do?
- Lets you build, run, and schedule custom command-line automations with one click.
- Does Automata need to stay open to run scheduled automations?
- No, scheduled runs use macOS launchd and fire even when the app is closed.
- Does Automata collect or send any data?
- No, everything runs and stays local on your Mac.
- How do I report a bug or request a feature?
- Open an issue on the GitHub repo’s Issues tab or join the Discord server.
